# Lab-PDI-VA

Desarrollado por estudiantes de la ESCOM, **Lab-PDI-VA** es una aplicación de escritorio educativa centrada en el **Procesamiento Digital de Imágenes (PDI)** y la **Visión Artificial (VA)**.

## Descripción

Este repositorio ofrece una colección completa de algoritmos de procesamiento de imágenes integrados en una interfaz gráfica intuitiva, diseñada para facilitar el aprendizaje, la experimentación y la exploración de técnicas de visión por computador.

## Características

La aplicación permite realizar prácticas directas con diversas operaciones clásicas de procesamiento de imágenes:

- Detección de bordes
- Transformaciones morfológicas
- Filtrado espacial
- Segmentación de imágenes
- Ecualización de histogramas
- Conteo de objetos
