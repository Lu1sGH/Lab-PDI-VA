Desarrollado por estudiantes de la ESCOM, <i>Lab-PDI-VA</i> es una aplicación de escritorio educativa centrada en el Procesamiento Digital de Imágenes (PDI) y la Visión Artificial (VA).
Este repositorio ofrece una colección completa de algoritmos de procesamiento de imágenes integrados en una interfaz gráfica intuitiva, diseñada para facilitar el aprendizaje, la experimentación y la exploración de técnicas de visión por computador.

La aplicación permite realizar prácticas directas con diversas operaciones clásicas de procesamiento de imágenes, como detección de bordes, transformaciones morfológicas, filtrado espacial, segmentación de imágenes, ecualización de histogramas y conteo de objetos.

Para más información, consulta la documentación generado en https://deepwiki.com/Lu1sGH/Lab-PDI-VA/1-home
